## Local Development

To view the website locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. Or serve it using a local web server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```
